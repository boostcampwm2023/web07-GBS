# 🔎 프로젝트 소개

```
GBS에서는 아프리카 TV, 트위치와 같이 실시간으로 인터넷 방송을 하거나 볼 수 있습니다.

스트리머는 OBS를 사용해서 GBS로 실시간 방송을 송출할 수 있습니다.

또한, 시청자는 방송을 보면서 채팅을 통해 다른 사람들과 실시간으로 소통할 수 있습니다.
```
