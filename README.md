# web07-GBS (금쪽이들의 방송 플랫폼)

## 프로젝트 개요

"금쪽이들의 방송 플랫폼"은 트위치, 아프리카와 같은 실시간 방송 스트리밍 서비스입니다. 이 프로젝트는 스트리밍 기술의 내부 작동 방식을 탐구하고, 기술적 도전을 통해 팀원들의 성장을 목표로 합니다.
